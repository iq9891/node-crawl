# node-crawl #
node抓取页面信息

## 使用方法 ##

1. 第一步就是在当前路径打开cmd
2. 输入如下代码
<pre>npm install</pre>
3. 输入如下代码
<pre>node app</pre>

## 功能 ##
1. app.js 用http.request抓取页面
2. app1.js 用http.get抓取页面
3. app2.js 用http.get抓取页面，并用cheerio进行节点筛选
4. geitInfo.js (1)从搜狗搜索中下载吆喝科技公众账号的图片地址，文章标题，文章链接，存入 ext.json文件；(2)把图片从网上下到本地存入 knowledgeImg 文件夹中
